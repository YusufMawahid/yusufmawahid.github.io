dalam aplikasi ini terdapat 3 hak akses yaitu Super Admin, Admin, dan User yang mempunyai tugas masing-masing, yaitu :
- Admin Super: pada hak akses ini, pada menu data pelamar super admin dapat mengaktifkan atau menonaktifkan pengguna jika belum melakukan transaksi oleh perusahaan, pada menu Tahapan Rekrutmen admin mengatur lulus atau tidaknya Pengguna untuk lanjut ke tahap selanjutnya melalui Gmail, Admin dapat mengelola artikel atau blog yang dapat dilihat oleh umum
- Admin: di hak akses ini semua fungsinya sama dengan Super Admin, hanya bedanya di menu Billing, admin hanya dapat melihat dan tidak diberikan akses untuk mengedit atau menghapus
- Pengguna: di hak akses ini, pengguna dapat mengisi data dari formulir yang disediakan, kemudian melakukan pembayaran ke perusahaan agar dapat mengikuti tes, dan dapat menerimanya melalui gmail, pada menu Hasil Tes pengguna dapat melihat hasil tes secara pribadi, dan di menu preview Pengguna dapat melihat data pribadi mereka dan mengunduh surat pernyataan yang telah terdaftar
